# Dealer Evaluation System

## Project Name
**Dealer Evaluation System (DES)**

## Overview
A microservices-based system for evaluating and comparing dealer pricing across multiple automotive products. The system consists of the following microservices:

- **Product Details Microservice** - Manages product information and specifications
- **Dealer Pricing Microservice** - Handles dealer-specific pricing and discounts
- **Dealer Evaluation Frontend** - React-based web interface for end users

## Microservices Architecture
| Microservice | Technology | Port |
|-------------|------------|------|
| Product Details | Node.js/Express | 4000 |
| Dealer Pricing | Node.js/Express | 5000 |
| Frontend | React.js | 3000 |
